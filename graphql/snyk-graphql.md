# Snyk GraphQL API

Snyk is a developer security platform for finding and fixing vulnerabilities in code, dependencies, containers, and IaC. The API covers project scanning, vulnerability data, dependency graphs, license compliance, and CI/CD integration.

**Endpoint:** No native GraphQL endpoint — conceptual schema derived from Snyk REST API
**Documentation:** https://docs.snyk.io/snyk-api

- Reference: https://apidocs.snyk.io/
- Schema: https://github.com/api-evangelist/snyk
